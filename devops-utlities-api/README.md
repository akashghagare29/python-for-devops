# Internal DevOps Utilities API

## Aim

This project aims to deliver production grades Utilities APi to internal teams:

- AWS Resources API
- Metrics
- Log Analysis

## Usage

`git clone <git_repo_url>`

### Setup Python Environment

`python3 -m venv venv`

`source venv/bin/activate`

### Install requirements packages/dependency
`pip install -r requirements.txt`

### Run the application
`python main.py`
